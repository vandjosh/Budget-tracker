# 💸 BudgetFlow

A modern personal budgeting dashboard built with HTML, CSS, and JavaScript.

Track your monthly spending, manage recurring bills, and visualize your financial trends — all in a clean, interactive UI.

---

## 🚀 Features

- 💰 Set a monthly budget
- 🧾 Add and track expenses
- 🔁 Recurring bills with automatic impact on remaining balance
- 📊 Category-based spending breakdown (doughnut chart)
- 📈 Monthly spending trend visualization
- 🔄 Automatic rollover of unused funds
- ✏️ Edit & delete recurring bills
- 💾 Data saved locally (no account required)

---

## 🧠 How It Works

- All data is stored in your browser using `localStorage`
- Each user/device has its own independent data
- No backend or database required

---

## 📸 Dashboard Overview

- **Remaining Balance** shows real available funds after expenses + upcoming bills
- **Trend Chart** shows how spending accumulates throughout the month
- **Category Chart** breaks down spending by category

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom UI styling)
- Vanilla JavaScript
- Chart.js (for visualizations)
